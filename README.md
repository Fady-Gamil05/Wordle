# Wordle 
A Java recreation of the popular word-guessing game Wordle.

## How it Works
- Player has 6 attempts to guess a 5-letter word
- After each guess, letters are colour-coded:
  - 🟩 Correct letter, correct position
  - 🟨 Correct letter, wrong position
  - ⬜ Letter not in the word

## Tech Stack
Java

## How to Run
1. Clone the repo
2. Open in IntelliJ IDEA or any Java IDE
3. Run Wordle.kt
